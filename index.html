Notes

Target color values must contain integers from 0 to 16.

Input rows do not need to be sorted by x. They will be sorted automatically.

The Stop button cancels the current scan, although it may not stop immediately during heavy computation.
There is no resume function.

Attribution is optional, but I would appreciate it if you include a link to this page in your submitted code.
